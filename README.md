# ✨ Key Features

## 🔎 Instant Smart Search

Quickly locate forms by typing:

-   Form number → `ACG 1`
-   Description → `Cash Book`

Search results update instantly as you type.

------------------------------------------------------------------------

## 📂 Category Filters

Forms are organized into departmental categories:

-   💰 Accounts\
-   📑 Registration\
-   📬 Mail\
-   📘 IPO

This allows focused study during preparation.

------------------------------------------------------------------------

## 🧠 Memorize Mode (Active Recall)

Designed specifically for **exam preparation and memory training**.

When enabled:

-   Form descriptions are **blurred**
-   Hover or tap to **reveal the answer**
-   Works like **flashcards**

Active recall significantly improves long‑term memory.

------------------------------------------------------------------------

## 📱 Fully Mobile Friendly

The interface works smoothly on:

-   📱 Smartphones\
-   📟 Tablets\
-   💻 Desktop browsers

Perfect for quick revision anytime.

# 🎯 Purpose of the Project

India Post departmental exams require remembering **a large number of
forms and registers**.

This tool simplifies the process using:

-   structured categories
-   quick searching
-   flashcard-style memorization

Useful for preparation of:

-   **GDS → MTS departmental exams**
-   **Postman / Mail Guard exams**
-   **Internal departmental training**
-   **Daily operational familiarity with postal forms**
